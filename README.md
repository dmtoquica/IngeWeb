# IngeWeb
# Proyecto de Ingeniería Web

Aplicación web educativa desarrollada en HTML, CSS y JavaScript puro.
Guía completa del proyecto del semestre organizada por etapas.

## Estructura
- index.html → estructura principal
- css/style.css → estilos
- js/app.js → lógica e interactividad

## Uso Ejemplo
Abrir index.html en cualquier navegador.
No requiere servidor ni dependencias.

## Propósito Ejemplo
Uso académico – Ingeniería Web
